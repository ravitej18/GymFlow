## What was built

1. **BMI Meter UI Widget**:
   - Replaced the read-only numeric input and plain label with a custom segmented horizontal bar (`data-bmi-meter`).
   - Defined 6 WHO Asian/Indian standard weight classification zones: Underweight, Healthy, Overweight, Obese I, Obese II, and Obese III.

2. **Animated Cursor & Numeric Details**:
   - Renders a custom vertical cursor (`data-bmi-cursor`) that moves dynamically to the relative position of the member's calculated BMI on a linear scale of 10 to 40.
   - Outputs the numeric BMI value and categorization text dynamically matching the category zone color.

3. **Input Sync & Auto-Hiding**:
   - Stores the BMI value in a hidden input field (`initBmi`) so that the Firestore save payload continues to receive it.
   - Automatically hides the meter panel if either weight or height fields are empty or invalid.
