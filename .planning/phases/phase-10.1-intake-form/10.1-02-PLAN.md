<objective>
Replace the read-only BMI number input and plain text label with a color-coded horizontal
visual meter. The meter shows BMI position across six color zones using WHO Asian/Indian
thresholds. An animated cursor moves to the calculated position. The meter is hidden until
both weight and height have valid values.

After this plan:
- BMI field renders as a full-width visual bar with six color zones (blue → green → yellow → orange → red → dark-red)
- A white cursor with dark border animates to the correct position (CSS transition)
- The BMI value is displayed numerically above the bar in the zone color
- The zone label (e.g., "Healthy") is shown beneath the bar
- The bar is hidden (`class="hidden"`) until both weight and height yield a valid BMI
- The hidden `<input name="initBmi">` still feeds `formData()` so save logic is unchanged
- `updateBmi()` is fully rewritten to drive the new meter

The `bmiCategory()` function and thresholds are unchanged — only the display layer changes.
</objective>

```html

                <div class="bmi-zone bmi-zone--uw"  title="Underweight < 18.5"></div>
                <div class="bmi-zone bmi-zone--ok"  title="Healthy 18.5–22.9"></div>
                <div class="bmi-zone bmi-zone--ow"  title="Overweight 23–24.9"></div>
                <div class="bmi-zone bmi-zone--ob1" title="Obese I 25–29.9"></div>
                <div class="bmi-zone bmi-zone--ob2" title="Obese II 30–34.9"></div>
                <div class="bmi-zone bmi-zone--ob3" title="Obese III ≥ 35"></div>
```

**Acceptance:** No `<input name="initBmi" type="number">` exists in the rendered form. A `[data-bmi-meter]` wrapper is present.

### Task 2 — Rewrite updateBmi() to drive the meter (members.js bind)

**File:** `modules/members.js`

Find the entire existing `updateBmi` function and its surrounding setup:
```js
    const bmiLabel = form.querySelector("[data-bmi-label]");
    function updateBmi() {
      const val = calcBmi(form.initWeight.value, form.initHeight.value);
      form.initBmi.value = val;
      const cat = bmiCategory(val, form.gender.value);
      if (bmiLabel) {
        bmiLabel.textContent = cat ? cat.label : "";
        bmiLabel.style.color = cat ? cat.color : "";
      }
    }
```

Replace with:
```js
    const bmiMeter    = form.querySelector("[data-bmi-meter]");
    const bmiNumber   = form.querySelector("[data-bmi-number]");
    const bmiCatEl    = form.querySelector("[data-bmi-category]");
    const bmiCursor   = form.querySelector("[data-bmi-cursor]");
    const bmiHidden   = form.querySelector("[data-bmi-hidden]");

    function updateBmi() {
      const val = calcBmi(form.initWeight.value, form.initHeight.value);
      if (bmiHidden) bmiHidden.value = val;

      if (!val) {
        if (bmiMeter) bmiMeter.classList.add("hidden");
        return;
      }

      if (bmiMeter) bmiMeter.classList.remove("hidden");

      const cat = bmiCategory(val, form.gender.value);
      if (bmiNumber)  { bmiNumber.textContent = val; bmiNumber.style.color = cat ? cat.color : ""; }
      if (bmiCatEl)   { bmiCatEl.textContent = cat ? cat.label : ""; bmiCatEl.style.color = cat ? cat.color : ""; }

      // Cursor position: linear scale 10–40 (30-unit range)
      const pct = Math.min(Math.max((parseFloat(val) - 10) / 30 * 100, 0), 100);
      if (bmiCursor) bmiCursor.style.left = `${pct}%`;
    }
```

**Acceptance:**
- Entering weight=70, height=170 → BMI 24.2 shown, cursor in overweight (yellow) zone, label "Overweight"
- Entering weight=55, height=170 → BMI 19.0, cursor in healthy (green) zone, label "Healthy"
- Clearing weight field → meter hides
- Editing a member with initWeight=80, initHeight=175 → meter renders correctly on load
